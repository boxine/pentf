newPage

  • newPage(config: TaskConfig, chrome_args?: undefined | string[]): Page
    • Defined in browser_utils.js:47

    Launch a new browser with puppeteer, with a new page (=Tab). The browser is completely isolated from any other calls. Most interactions will be with the page, but you can get the browser using await page.browser();. For more information about the page object, see the puppeteer API documentation.

    example
    const page = await newPage(config);
await page.goto('https://example.org/');
await waitForText(page, 'More information');
await closePage(page);

    Parameters

    • config: TaskConfig

      The pentf configuration object.

    • Default value chrome_args: undefined | string[] = []

    Returns Page

    The puppeteer page handle.

    • Defined in browser_utils.js:812
